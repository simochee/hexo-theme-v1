## Install

このテーマを使うために、ベースでイカを実行してください。

```
yarn add hexo-renderer-pug
```

## `source`ディレクトリについて

`/path/to/theme/source`はテーマから見たルートディレクトリとなるが、この内容は`_src`の内容をGulpを通して動的に生成、または移動されたファイルのみとする。
